FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help
developers improve their frontend coding skills with projects in
HTML, Css, and Javascript. It's suitable for all levels and ideal for
portfolio building.

is Frontend Mentor free?

It is both free and premium

Can I use Frontend Mentor projects in my portfolio?

Definitely! Please do feel free to use whatever you build in your portfolio. Helping developers add professional-looking projects
to their portfolio was one of the reasons we created this
platform!

How can I get help if I'm stuck a challenge?

The best (and quickest) way to get help on a challenge is in our Discord server. There are thousands of other developers in
there, so it's a great place to ask questions.
